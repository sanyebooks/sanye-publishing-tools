# 图书卖点拆解 Plugin 试用授权条款

版本：0.1（试用稿）

> 本文件是商业试用模板，不构成法律意见。正式对外签署前，建议由法律专业人士根据你的主体、地区和交易方式审核。

## 1. 授权内容

授权方允许指定客户在约定试用期内，为内部图书编辑与营销评估目的使用“图书卖点拆解 Plugin”。授权为非独占、不可转让、不可再许可的有限试用权。

## 2. 允许的使用

- 由客户内部员工安装和使用本工具。
- 使用客户有权处理的书稿、样章和出版资料进行测试。
- 将生成结果用于内部讨论、修改和评估。
- 经人工审核后，将修改后的结果用于客户自有图书宣传。

## 3. 不允许的使用

- 向客户以外的个人或机构复制、转售、出租或公开发布 Plugin 源文件。
- 删除或修改作者、版本和授权标识后再次分发。
- 通过逆向整理、批量复制等方式制作实质相同的商业工具并对外提供。
- 处理客户无权使用的书稿、个人信息或商业秘密。
- 将未经人工核校的结果直接作为事实、广告承诺或法律依据发布。

## 4. 账号与安装数量

默认试用范围为一个客户主体、最多三名内部用户、最多三台设备。其他范围应另行书面约定。

## 5. 试用期限

默认试用期为收到试用包之日起14日。试用期满后，客户应停止使用并删除本地安装包；如需继续使用，应购买正式授权。

## 6. 书稿与数据

客户应自行确认所使用的 Codex、模型服务、企业账号及本地环境符合其书稿保密制度。授权方不因提供 Plugin 而当然取得客户书稿权利。若授权方需要接收书稿或结果用于定制，双方应另行约定数据处理、保存期限和删除方式。

## 7. 生成结果与人工审核

生成内容可能存在遗漏、误判、页码偏差或表达不适用。客户应由具备专业判断的编辑进行审核。授权方不保证销量、传播量、平台审核结果或商业收益。
