# 图书卖点拆解 Plugin｜出版社客户试用版

版本：`0.1.0`  
提供方：孙三野  
适用对象：出版社编辑、营销编辑、发行与新媒体团队

本工具读取 PDF、Word、样章、目录或粘贴的书稿资料，生成有书内依据、页码、营销优先级和目标人群圈层的图书卖点 Excel。

## 试用包内有什么

- `plugins/book-selling-points/`：可安装的 Codex Plugin
- `安装指南.md`：零技术安装步骤
- `编辑使用手册.md`：编辑部实际操作方法
- `examples/示例成品-图书营销卖点表.xlsx`：完整示例
- `试用授权条款.md`：客户试用边界
- `CHANGELOG.md`：版本更新记录

## 最快开始

1. 按《安装指南》安装 Plugin。
2. 在 Codex 新建任务并上传书稿。
3. 输入：“使用图书卖点拆解，把这本书拆成30条有页码依据的营销卖点表。”
4. 等待生成并下载 Excel。

## 推荐测试方式

首次试用请选择一本已出版或可以在内部测试的书稿。建议同时安排一名编辑审核结果，并记录“保留、修改、删除”的条目，这些反馈可用于后续定制出版社专属版本。

## 重要提示

- 正式用于腰封、详情页或宣传物前，请编辑复核人物名、数据和最终排版页码。
- 小说类结果可能包含剧透；工作簿会给出分阶段使用建议。
- 请遵守本单位关于未出版书稿、作者资料和商业机密的处理规定。
- 本试用包不是法律、版权或市场销量保证工具。
