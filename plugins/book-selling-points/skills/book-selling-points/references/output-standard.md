# 图书卖点表输出标准

## 人群圈层

- `核心层`：本书主题问题的一线承受者、购买者或强需求者。卖点优先命中他们的真实焦虑、行动需求和购买理由。
- `扩展层`：能够影响购买、形成专业背书、带来渠道传播或二次扩散的人群。
- `潜力层`：不一定因核心场景直接购买，但可能被价值观、身份、社会议题、职场或情绪议题吸引的人群。

卖点表中把标签写入 `针对的人群圈层`，再用一句话解释该人群为什么会被这条卖点打动。另建 `人群圈层说明` 工作表保存该书的具体圈层定义。

## 星级

- `超级加星`：强痛点、强购买理由、强传播表达、强书内支撑，适合首批投放、直播或详情页。
- `加星`：有明确传播潜力或能支撑内容矩阵，但不是第一优先级。
- 空白：补充卖点、长尾内容或章节覆盖。

不要把所有行标星。

## 小红书笔记

每条笔记都需要：网感化标题、具体钩子、书内证据、读者可带走的行动或判断，以及页码/章节线索。

在不同笔记间变化：

- 开头：反常识、场景、问题、案例、清单、写给某类人的信、评论区共鸣。
- 结构：短段观点、三点清单、前后对比、误区纠正、案例复盘、行动建议、金句拆解。
- 结尾：问题、保存分享、读者行动或自我反思。

不得重复同一套开头、三段式正文和结尾。

## 四张图提示词

每条内容恰好生成四张：

1. 封面图：支撑标题，明确标题留白和构图。
2. 配图1：用隐喻、对比、信息图或物件场景解释概念。
3. 配图2：呈现与书内证据相关的具体生活、家庭、学校、职场或社会场景。
4. 配图3：总结方法、清单、行动、章节线索或金句含义。

同一本书保持统一视觉气质，但要变化场景和图像形式，避免通用图库感。

## Excel格式

- 冻结表头并开启筛选。
- 所有长文本自动换行、垂直顶端对齐。
- 设置实用列宽和适中行高。
- 为 `星级` 和 `针对的人群圈层` 设置数据验证。
- 长文案和图像提示词放在独立工作表，不挤压主卖点表。
- 渲染并目视检查主表及至少一个内容页。
