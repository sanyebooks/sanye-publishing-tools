# 编辑使用手册

## 一、它适合做什么

- 新书选题会前的卖点初筛
- 营销方案中的购买理由梳理
- 腰封、详情页和发行资料的内容储备
- 小红书、公众号、短视频选题池
- 不同读者圈层的传播角度拆分

它输出的是“供编辑选择和修改的证据型初稿”，不是替代责任编辑的最终文案。

## 二、最好提供哪些材料

从高到低依次为：

1. 有稳定正文页码的完整 PDF
2. 完整 Word 书稿
3. 目录＋简介＋三个以上样章
4. 目录＋内容简介

材料越完整，卖点独特性和页码可靠性越高。扫描版 PDF 可能需要额外识别时间。

## 三、标准提示词

### 完整卖点表

> 使用图书卖点拆解，通读这本书，生成30条有书内依据的营销卖点 Excel。字段包含原标题、网感化标题、具体内容、星级、正文页码、人群圈层和圈层说明。不要用泛泛而谈的套话。

### 小说预热版

> 使用图书卖点拆解，为这本小说生成30条卖点。重点提炼世界观规则、人物困境和高概念情节；预热期使用，不披露关键死亡和最终结局。另列深度传播期可用的含剧透卖点。

### 非虚构版

> 使用图书卖点拆解，为这本非虚构作品提炼30条卖点。优先提炼独有观点、方法、案例、数据、清单和反常识结论，所有卖点必须有页码依据。

### 小红书内容版

> 在卖点表基础上，选择10条最适合小红书的卖点，分别生成不重复的小红书笔记和封面图、3张配图的AI创作提示词。

## 四、编辑审核方法

建议新增三列：`编辑决定`、`修改意见`、`计划渠道`。

- 保留：确实是本书独有、能形成购买理由。
- 修改：证据成立，但标题不符合本社调性。
- 删除：重复、剧透过强、目标人群不合适或不构成购买理由。

特别核查：

- 人物名、作者经历、专有名词是否准确
- 页码是否与最终出版版本一致
- 标题是否超出原文证据
- 数据是否缺少上下文
- 是否泄露小说关键结局
- 是否符合宣传合规与版权要求

## 五、建议的试用验收标准

- 30条中至少18条服务核心读者
- 至少20条能明确指回书内具体内容
- 不出现编造的原文、数据和页码
- 同义重复项不超过3条
- 编辑无需重写即可保留的条目达到30%以上
- Excel 清晰可读，可以直接进行内部筛选

## 六、如何反馈以获得定制版

请将编辑审核后的 Excel 返回给服务方，并说明：

- 本社常用文案语气
- 绝对不能出现的词
- 重点渠道
- 典型读者画像
- 过去表现好的宣传案例
- 希望固定增加或删除的字段

这些反馈可以被整理为出版社专属规则，但不应包含无授权的作者隐私或第三方商业秘密。
